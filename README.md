NODEPOP

Aplicación web desarrollada con Node, Express y MongoDB.

Permite a los usuarios (crear, ver, eliminar) con autenticación mediante sesión.

Funcionalidades:

- Registro de usuarios
- Login 
- Crear productos 
- Listado de productos del usuario
- Eliminación de productos (sólo propios)
- Filtros:
       - Nombre
       - Tag
       - Precio (min/max)


Autenticación:

Los usuarios han de registrarse e iniciar sesión para acceder a la app.

Cada usuario puede:
- Ver sus propios productos
- Eliminar sus propios productos
